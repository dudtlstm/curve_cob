## 🎯 Commit Convention

- ✨ Feat : 새로운 기능 추가
- 🎨 fix : 코드의 구조/형태 개선
- 🐛 bug : 버그 수정
- 🚑 ambulance : 긴급 수정
- 📑 docs : 문서 수정
- 💄 style : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- ♻️ refactor: 코드 리펙토링
- 💡 bulb : 주석 추가/수정
- 🧪 test: 테스트 코드, 리펙토링 테스트 코드 추가
- 🛠️ chore : 빌드 업무 수정, 패키지 매니저 수정
